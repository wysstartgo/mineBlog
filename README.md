# myBlog
一步步搭建个人博客系统


## 安装
 git clone https://github.com/wysstartgo/myBlog.git
## 安装包依赖
 npm install
 
## 使用cnpm安装包依赖更快，首先安装cnpm

npm install -g cnpm --registry=https://registry.npm.taobao.org
cnpm install

## 运行
### 运行发开环境

npm run dev

本地访问http://localhost:3000

### 打包部署，打包部署必须在服务器端
首先打包编译项目

npm run build
运行项目

npm run start
